# MasterSlaveSolution

Master-slave система на основе протокола TCP для сравнения двух текстов по смыслу на основе составления соответствующих матриц данных.
Клиент отправляет несколько текстов. Master-узел помещает их в очередь и равномерно распределяет между slave-узлами. Slave-узлы считывают частоту появления слов согласно запроса и отправляют вектор ответа и время обработки. Сервер определяет вероятность наиболее близких по смыслу текстов. Для передачи данных используется протокол TCP.
